	% -------------------- ABSTRACT --------------------
	\chapter*{Abstract}
	
	The rapid growth of Internet of Things (IoT) technology has enabled smart communication between devices. This project presents an IoT-based Bit Mapping system that demonstrates how binary data can be represented and visualized using connected devices. The system uses a microcontroller to process binary input and map each bit to LEDs. The data is also transmitted to an IoT platform for real-time monitoring. The project highlights the integration of digital electronics with IoT and provides a practical approach for understanding binary systems.

	\chapter{Introduction}

		\item Enable real-time monitoring

	\section{Limitations}
